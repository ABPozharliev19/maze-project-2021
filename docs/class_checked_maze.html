◆ checkMaze()

void CheckedMaze::checkMaze ( )

The function recursively checks a coordinate, that is not a wall, then chooses a random neighbour that has not been visited yet, and repeats the process until it gets to the end of the maze.
If there isn't a neighbour that is unchecked, backtracks to a coordinate that has any unvisited neighbour(s) left.
Else if there aren't any unchecked cells it generates a new maze and repeats the process, until there is a solvable maze.
